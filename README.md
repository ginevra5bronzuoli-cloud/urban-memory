# AURA

AURA is a prototype Adaptive Cognitive Assistant: a soft, wellness-oriented personal operating system that helps organize daily life around energy, mental load, routines, nutrition, and gentle planning.

## What is included

- Animated AI brain as the central assistant.
- Dynamic daily planner with energy, duration, category, and balance checks.
- Natural language input for tasks, meals, mood, pets, plants, and delays.
- Nutrition & Energy section with soft macro and hydration estimates.
- Personal Care section for movement, hygiene, and mental wellbeing.
- Diary section with free writing and gentle insights.
- Adaptive UI modes for calm, focus, and recovery.

## Run locally

Open `app/index.html` in a browser, or serve the `app` folder with any static server.
